QuickChat

QuickChat is a non-profit experimental chat app by Topic Duo. Connect randomly and chat with fun, short bursts!

Web Model Preview

This preview uses an AI instead of real users, showing off the app’s interface, themes, and timers. Everything is still a work in progress!

Features

  • Random conversation themes every round
  • Timer showing time left per theme
  • AI chat simulating real users
  • Reset, Quit, and Continue controls
  • Short, casual, fun chat style

⚠️ Preview only — features may change before full release.
